# Elevator Simulation

An experimental elevator simulation for experimenting with object-oriented and Clean Architecture principles.


# Assumptions

* number of elevators? 1
* top floor? 5
* lowest floor? -1
* single request button on each floor, not up/down

Running tests
----

```sh
$ bin/rspec --format doc
```
